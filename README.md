# Anime-tracker
----------------
**Описание**:  Это приложение позволяет отслеживать проогресс просмотра аниме. Возможна работа с MyAnimeList, с базой двнных или с файлом.
 - **Технологический стек**: JavaFX, SQLite, MyAnimeList API.
 - **Статус**:  1.0.
